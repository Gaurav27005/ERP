## 🚀 Quick Start

```bash
# 1. Backend
cd backend && npm install
npm run seed        # seed database once
npm run dev         # runs on http://localhost:5000

# 2. Frontend (new terminal)
cd frontend && npm install
npm start           # runs on http://localhost:3000
```

Requirements: Node.js v18+, MongoDB running locally

## ✨ Features

| Module | What it does |
|--------|-------------|
| **Dashboard** | Role-based stats, notices, upcoming drives & interviews |
| **Attendance** | Faculty marks own subjects only · Edit past records · All records visible · Student sees own % with warnings |
| **Study Material** | Upload via URL or device file · Type tabs with correct counts · Uploader + Admin can edit/delete |
| **Placements** | Full drive lifecycle · Eligibility check · Student registration · TPO create/edit/delete |
| **Mock Interviews** | Schedule sessions · Rating sliders 1-10 · Full feedback history |
| **Leaderboard** | LeetCode/CodeChef/Codeforces/HackerRank/GitHub · Auto-score · Podium for top 3 |
| **Students** | Table + Card view · Search by name/roll/PRN · Filter by dept/year |
| **Faculty** | Directory by department · Contact info |
| **Notices** | Type tabs with accurate counts · Priority color coding · Notification on post |
| **Notifications** | Bell with unread badge · Mark read/all-read · Auto-refresh |
| **Profile** | Edit info · Change password |

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|

| Database | MongoDB |
| Design | Custom CSS design system — DYP Navy (#0d1b4b) + Gold (#c8960c) |
| Fonts | Sora + JetBrains Mono |
